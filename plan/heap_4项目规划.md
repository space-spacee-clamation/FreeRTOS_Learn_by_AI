# heap_4 内存分配器学习项目规划

## 项目概述

heap_4是FreeRTOS提供的高级内存分配器，在heap_2的基础上进行了重大改进。本项目将深入学习heap_4的工作原理、使用方法和最佳实践。

**核心特点：**
- 支持相邻空闲块自动合并（coalescing）
- 使用首次适配算法（First Fit）
- 线程安全实现
- 提供完整的内存统计功能
- 支持内存块分割
- 有效减少内存碎片化

**与heap_2的关键区别：**
- heap_2使用最佳匹配算法，heap_4使用首次适配算法
- heap_4在释放时自动合并相邻空闲块
- heap_4的空闲块链表按地址排序，便于合并
- heap_4抗碎片化能力更强

## 学习目标

1. 理解heap_4的工作原理和算法实现
2. 掌握首次适配算法与最佳匹配算法的区别
3. 学习相邻空闲块合并机制
4. 掌握heap_4的使用场景和最佳实践
5. 理解如何通过heap_4减少内存碎片化
6. 学会使用vPortGetHeapStats()获取详细内存统计

## 与heap_2的关键对比

| 特性 | heap_2 | heap_4 |
|------|--------|--------|
| 算法 | 最佳匹配（Best Fit） | 首次适配（First Fit） |
| 合并 | 仅合并到下一个块 | 自动合并前后相邻块 |
| 空闲块排序 | 按大小排序 | 按地址排序 |
| 碎片化 | 可能产生小碎片 | 有效减少碎片 |
| 查找速度 | 较慢（需遍历所有） | 较快（找到第一个合适的） |
| 适用场景 | 分配大小较固定 | 频繁分配/释放，大小多变 |

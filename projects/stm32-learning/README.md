# STM32平台FreeRTOS渐进式学习项目集合

这是一个边学边测试的STM32 FreeRTOS学习实践仓库，按照学习进度逐步深入理解FreeRTOS的各个核心功能模块。

## 目录结构与命名规范

目录采用**章节编号+模块名称**的命名方式，体现渐进式学习路径：

- `01-getting-started/` - 入门项目，环境搭建与第一个FreeRTOS任务
- `02-core-kernel/` - 核心内核功能（任务创建、调度、队列、信号量、互斥锁等）
- `03-memory-management/` - ⭐ 堆内存管理（重点研究）- 各种内存分配算法的对比与实践
- `04-software-timers/` - ⭐ 软件定时器（重点研究）- 定时器应用与精度测试
- `05-stream-buffers/` - ⭐ 流缓冲区（重点研究）- 大数据流处理与中断通信

每个编号预留了多个子项目的扩展空间，可以根据需要在模块目录下创建具体的练习项目。

## 项目内容要求

每个具体项目应该包含：

- **源代码**：完整的可编译项目代码
- **构建文件**：Makefile或CMake配置（根据开发环境选择）
- **说明文档**：README.md，记录学习要点、实验目的、测试结果和心得体会

## 重点研究方向

本集合特别关注以下三个核心模块的深入实践：

1. **内存管理** - 理解FreeRTOS不同内存分配方案（heap_1到heap_5）的优缺点和适用场景
2. **软件定时器** - 掌握定时器回调设计、精度测试、资源共享处理
3. **流缓冲区** - 探索流缓冲区在中断与任务通信中的高效应用，对比队列的使用差异

## 💻 学习项目

### STM32平台学习项目
项目入口：[projects/stm32-learning/](projects/stm32-learning/)

- [projects/stm32-learning/README.md](projects/stm32-learning/README.md) - STM32学习项目介绍与目录规范

项目采用**章节编号+模块名称**的渐进式组织结构：

| 模块 | 说明 |
|------|------|
| `01-getting-started/` | 入门项目，环境搭建与第一个FreeRTOS任务 |
| `02-core-kernel/` | 核心内核功能（任务创建、调度、队列、信号量、互斥锁等） |
| `03-memory-management/` | ⭐ 堆内存管理（重点研究）- 各种内存分配算法的对比与实践 |
| `04-software-timers/` | ⭐ 软件定时器（重点研究）- 定时器应用与精度测试 |
| `05-stream-buffers/` | ⭐ 流缓冲区（重点研究）- 大数据流处理与中断通信 |

每个子项目包含完整源代码、构建文件和说明文档。

## 📌 重要原则

- **渐进式学习**：按照阶段逐步深入，不要跳跃
- **文档代码分离**：文档放doc/，代码放projects/
- **经验沉淀**：解决问题后及时总结存入doc/experiences/对应分类
- **Git规范**：严格遵守[doc/Git规范.md](doc/Git规范.md)的分支命名和提交规范
- **保持简洁**：每个文档聚焦一个主题，删除过时错误内容

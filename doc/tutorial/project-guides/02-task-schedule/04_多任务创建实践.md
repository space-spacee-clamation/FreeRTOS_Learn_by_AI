# 多任务创建实践

## 创建多个任务的步骤

### 回顾：单个任务创建

```c
xTaskCreate(
    vTaskFunction,     // 任务函数
    "TaskName",        // 任务名称
    128,               // 栈大小
    NULL,              // 任务参数
    1,                 // 优先级
    NULL               // 任务句柄
);
```

### 创建多个任务

创建多个任务就是多次调用xTaskCreate()：

```c
// 创建任务1
xTaskCreate(vTask1, "Task1", 128, NULL, 1, NULL);

// 创建任务2
xTaskCreate(vTask2, "Task2", 128, NULL, 2, NULL);

// 创建任务3
xTaskCreate(vTask3, "Task3", 128, NULL, 3, NULL);
```

## 本项目的多任务创建

### 任务设计

我们创建了3个任务，每个任务有不同的优先级：

| 任务 | 优先级 | 周期 | 说明 |
|-----|-------|------|------|
| HighTask | 3 | 200ms | 高优先级，运行频率最高 |
| MediumTask | 2 | 300ms | 中优先级 |
| LowTask | 1 | 500ms | 低优先级 |

### 完整的创建代码

```c
int main(void)
{
    // 初始化串口
    vUARTInit();

    // 输出信息
    vPrintString("正在创建任务...\r\n");

    // 创建低优先级任务
    BaseType_t xResult1 = xTaskCreate(
        vLowPriorityTask,   // 任务函数
        "LowTask",          // 任务名称
        128,                // 栈大小
        NULL,               // 任务参数
        1,                  // 优先级 1
        NULL                // 任务句柄
    );

    // 创建中优先级任务
    BaseType_t xResult2 = xTaskCreate(
        vMediumPriorityTask,
        "MediumTask",
        128,
        NULL,
        2,                  // 优先级 2
        NULL
    );

    // 创建高优先级任务
    BaseType_t xResult3 = xTaskCreate(
        vHighPriorityTask,
        "HighTask",
        128,
        NULL,
        3,                  // 优先级 3
        NULL
    );

    // 检查是否都创建成功
    if((xResult1 == pdPASS) && (xResult2 == pdPASS) && (xResult3 == pdPASS))
    {
        vPrintString("所有任务创建成功！\r\n");
    }
    else
    {
        vPrintString("错误：任务创建失败！\r\n");
        for(;;);  // 死循环
    }

    // 启动调度器
    vTaskStartScheduler();

    // 只有调度器启动失败才会到这里
    for(;;);
    return 0;
}
```

## 任务函数的编写

### 高优先级任务函数

```c
void vHighPriorityTask(void *pvParameters)
{
    // 防止编译器警告
    (void)pvParameters;

    uint32_t ulCounter = 0;

    vPrintString("[High] 高优先级任务启动！\r\n");

    for(;;)
    {
        // 输出运行信息
        vPrintString("[High] 运行 #");
        vPrintNumber(ulCounter);
        vPrintString("\r\n");

        ulCounter++;

        // 阻塞200ms
        vTaskDelay(pdMS_TO_TICKS(200));
    }
}
```

## 观察和实验

### 实验1：观察默认输出

编译运行项目，观察输出顺序：

```
[High] 高优先级任务启动！
[High] 运行 #0
[Medium] 中优先级任务启动！
[Medium] 运行 #0
[Low] 低优先级任务启动！
[Low] 运行 #0
[High] 运行 #1
[Medium] 运行 #1
[High] 运行 #2
[Low] 运行 #1
...
```

**你会看到**：
- HighTask运行频率最高（每200ms一次）
- MediumTask其次（每300ms一次）
- LowTask最少（每500ms一次）

### 实验2：修改优先级

尝试修改任务优先级，观察变化：

```c
// 修改前
xTaskCreate(vLowPriorityTask, "LowTask", 128, NULL, 1, NULL);
xTaskCreate(vMediumPriorityTask, "MediumTask", 128, NULL, 2, NULL);
xTaskCreate(vHighPriorityTask, "HighTask", 128, NULL, 3, NULL);

// 修改后：把LowTask设为最高优先级
xTaskCreate(vLowPriorityTask, "LowTask", 128, NULL, 3, NULL);  // 改为3
xTaskCreate(vMediumPriorityTask, "MediumTask", 128, NULL, 2, NULL);
xTaskCreate(vHighPriorityTask, "HighTask", 128, NULL, 1, NULL);  // 改为1
```

**预期结果**：
- LowTask现在运行频率最高
- HighTask运行得最少

### 实验3：相同优先级

把所有任务设为相同优先级：

```c
xTaskCreate(vLowPriorityTask, "LowTask", 128, NULL, 2, NULL);
xTaskCreate(vMediumPriorityTask, "MediumTask", 128, NULL, 2, NULL);
xTaskCreate(vHighPriorityTask, "HighTask", 128, NULL, 2, NULL);
```

**预期结果**：
- 三个任务按时间片轮转
- 运行顺序可能是：High → Medium → Low → High → ...

### 实验4：不调用vTaskDelay()

修改HighTask，不调用vTaskDelay()：

```c
void vHighPriorityTask(void *pvParameters)
{
    (void)pvParameters;
    uint32_t ulCounter = 0;

    vPrintString("[High] 高优先级任务启动！\r\n");

    for(;;)
    {
        // 注意：不调用vTaskDelay()！
        vPrintString("[High] 运行 #");
        vPrintNumber(ulCounter);
        vPrintString("\r\n");

        ulCounter++;

        // 没有vTaskDelay()！
    }
}
```

**预期结果**：
- HighTask永远运行
- Medium和Low任务永远得不到运行
- 这就是"任务饿死"

## 调试技巧

### 技巧1：输出任务状态

在任务中输出更多信息：

```c
void vTaskFunction(void *pvParameters)
{
    for(;;)
    {
        vPrintString("任务运行，当前计数器 = ");
        vPrintNumber(ulCounter);
        vPrintString("，系统时间 = ");
        vPrintNumber(xTaskGetTickCount());
        vPrintString("\r\n");

        vTaskDelay(pdMS_TO_TICKS(100));
    }
}
```

### 技巧2：使用任务句柄

获取任务句柄，用于调试：

```c
TaskHandle_t xTaskHandle = NULL;

xTaskCreate(
    vTaskFunction,
    "Task",
    128,
    NULL,
    1,
    &xTaskHandle  // 获取任务句柄
);

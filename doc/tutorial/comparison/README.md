## 快速参考

### 同步机制快速选择

| 需求 | 推荐方案 |
|------|---------|
| 一对一同步 + 高性能 | 任务通知 |
| 共享资源保护 | 互斥信号量 |
| 多条件等待 | 事件组 |
| 简单通用 | 二值信号量 |

---

### 通信机制快速选择

| 需求 | 推荐方案 |
|------|---------|
| 多对多通信 | 队列 |
| 一对一 + 连续数据流 | 流缓冲区 |
| 一对一 + 变长消息 | 消息缓冲区 |

---

### 内存分配器快速选择

| 需求 | 推荐方案 |
|------|---------|
| 通用场景（推荐） | heap_4 ⭐⭐⭐⭐⭐ |
| 启动一次性分配 | heap_1 |
| 多RAM区域 | heap_5 |
| 快速原型 | heap_3 |

## 使用建议

1. **先看对应对比文档**: 在选择具体机制前，先阅读对应的对比指南
2. **参考决策流程图**: 每个文档都包含决策流程图，帮助快速选择
3. **看实际案例**: 每个文档都有实际应用案例，可直接参考
4. **关注最佳实践**: 遵循文档中的最佳实践建议
